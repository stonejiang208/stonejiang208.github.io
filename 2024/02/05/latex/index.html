欧拉公式

欧拉公式是数学中的一项重要公式，表示了复数、指数和三角函数之间的关系。它的表达式如下：

eix=cos⁡x+isin⁡xe^{ix}=\cos x + i\sin x eix=cosx+isinx

这个公式展示了虚数单位 iii 、自然对数的底 eee、余弦函数 cos⁡\coscos 和正弦函数 sin⁡\sinsin 之间的美妙关系。欧拉公式在数学、物理和工程等领域广泛应用，是一种极具深刻意义的数学表达式。

正态分布

正态分布（高斯分布）是自然界中许多现象的分布，其概率密度函数（PDF）表示为：

f(x;μ,σ)=1σ2πe−(x−μ)22σ2f(x; \mu, \sigma) = \frac{1}{\sigma \sqrt{2\pi}} e^{-\frac{(x - \mu)^2}{2\sigma^2}} f(x;μ,σ)=σ2π​1​e−2σ2(x−μ)2​

其中，μ\muμ 是均值，σ\sigmaσ 是标准差。

泊松分布

泊松分布用于描述单位时间或空间内某事件发生次数的概率分布，其概率质量函数（PMF）为：

P(X=k)=λke−λk!P(X = k) = \frac{\lambda^k e^{-\lambda}}{k!} P(X=k)=k!λke−λ​

其中，λ\lambdaλ 是事件发生的平均次数。

二项分布

二项分布适用于只有两种结果的离散随机变量，其概率质量函数为：

P(X=k)=(nk)pk(1−p)n−kP(X = k) = \binom{n}{k} p^k (1-p)^{n-k} P(X=k)=(kn​)pk(1−p)n−k

其中，nnn 是试验次数，kkk 是成功的次数，ppp 是成功的概率。

指数分布

指数分布描述等待时间或间隔时间的概率分布，其概率密度函数为：

大数定理

大数定理是概率论中的一个基本定理，描述了随机变量的均值在样本数量趋于无穷大时稳定地收敛到其期望值的现象。

切比雪夫大数定理

设 X1,X2,…,XnX_1, X_2, \ldots, X_nX1​,X2​,…,Xn​ 是独立同分布的随机变量序列，具有期望值 μ\muμ 和方差 σ2\sigma^2σ2。定义样本均值为 Xˉn=1n∑i=1nXi\bar{X}_n = \frac{1}{n} \sum_{i=1}^{n} X_iXˉn​=n1​∑i=1n​Xi​，那么对于任意正数 ϵ>0\epsilon > 0ϵ>0，切比雪夫不等式给出：

P(∣Xˉn−μ∣≥ϵ)≤σ2nϵ2P\left(\left|\bar{X}_n - \mu\right| \geq \epsilon\right) \leq \frac{\sigma^2}{n\epsilon^2} P(∣∣∣​Xˉn​−μ∣∣∣​≥ϵ)≤nϵ2σ2​

大数定理的主要思想是，随着样本数量 nnn 的增大，样本均值 Xˉn\bar{X}_nXˉn​ 以概率 1 收敛到期望值 μ\muμ。即：

lim⁡n→∞P(∣Xˉn−μ∣≥ϵ)=0\lim_{n \to \infty} P\left(\left|\bar{X}_n - \mu\right| \geq \epsilon\right) = 0 n→∞lim​P(∣∣∣​Xˉn​−μ∣∣∣​≥ϵ)=0

这说明在大样本的情况下，样本均值趋于稳定，逼近真实的期望值。

证明

利用切比雪夫不等式：

P(∣Xˉn−μ∣≥ϵ)≤σ2nϵ2P\left(\left|\bar{X}_n - \mu\right| \geq \epsilon\right) \leq \frac{\sigma^2}{n\epsilon^2} P(∣∣∣​Xˉn​−μ∣∣∣​≥ϵ)≤nϵ2σ2​

当 nnn 趋于无穷大时，右侧趋于零，证明完成。

这证明了大数定理的一个形式。注意，大数定理有多种形式和变体，取决于随机变量序列的性质和收敛的方式。这只是其中一种简单的表达方式。
